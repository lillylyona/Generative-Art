About

This banner uses Perlin noise to generate smooth, organic wave motion. Instead of designing a fixed image, the system draws multiple flowing lines every frame, producing an evolving composition.

Technique

  • Generative method: Randomness & Noise
  • Output: Layered wave lines
  • Behaviour: Continuous evolution over time

Interaction Controls

  • Mouse X → changes noise scale (smooth ↔ chaotic)
  • Mouse Y → changes wave height (subtle ↔ dramatic)
  • Click → regenerate (new seed)

Why interaction matters

The user doesn’t place shapes manually. Instead, their movement influences the parameters of the generative system in real time, keeping the artwork autonomous while still responsive.

Build Notes

This page is fully responsive. The p5 canvas automatically resizes to the hero banner width and height.

Tip: run with a local server (VS Code Live Server) so scripts load properly.
